# Create icon-512.png

You need to create a 512x512px icon for the PWA to work properly.

## Quick Options:

### Option 1: Resize Existing Icon (Easiest)
1. Open `public/icon-192.png` in any image editor
2. Resize to 512x512px (maintain aspect ratio)
3. Save as `public/icon-512.png`

### Option 2: Use Online Tool
1. Go to: https://www.pwabuilder.com/imageGenerator
2. Upload your `icon-192.png`
3. Download the 512x512 version
4. Save as `public/icon-512.png`

### Option 3: Create New Design
Use any design tool (Figma, Photoshop, Canva) to create:
- Size: 512x512px
- Format: PNG
- Design: Use your app's theme colors (#F4A6C1, #E8979D)
